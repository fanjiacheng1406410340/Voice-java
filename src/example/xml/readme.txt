============================================Voice HTTP XML  格式===================================================================

1、创建项目
2、引入jar
相关jar列表：（libs目录下）
ctc-smscloud-voice-xmlhttp.jar
commons-codec-1.5.jar
commons-httpclient-3.1.jar
commons-logging-1.1.1.jar
3、调用客户端示例，路径：java\voice_http\example\xml\VoiceHttpXMLClient.java

注：本示例支持 jdk1.6 或以上版本
